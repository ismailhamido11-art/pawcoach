1. **Lire le fichier** avec Read() avant toute modification
2. **Pour chaque import inutilise signale** :
   - Grep le symbole dans le fichier pour confirmer qu'il n'est NULLE PART dans le corps (pas en JSX, pas dans un callback, pas dans une condition)
   - Si confirme inutilise → supprimer la ligne d'import ou le symbole dans l'import destructure
   - Si import partiel (ex: `import { A, B, C }` ou seul B est inutilise) → retirer uniquement B de la liste
3. **Pour chaque variable/fonction non utilisee signale** :
   - Prefixer avec `_` si c'est un parametre de fonction requis par une signature (ex: `(_, index)`) — ne PAS supprimer les params requis par des callbacks
   - Supprimer la declaration si la variable/fonction n'est vraiment pas utilisee et n'est pas exportee
   - NE PAS supprimer une variable si elle est exportee (elle pourrait etre importee ailleurs)

